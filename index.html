Projet Académique // Agent IA Conversationnel

PeepChat — Système Conversationnel

Python SpeechBrain ECAPA-TDNN OpenAI Whisper LLaMA 3.1 (Groq) GAN OpenCV

PeepChat est un agent d'échange intelligent combinant traitement automatique de la parole, biométrie vocale et rendu graphique adaptatif. Le système effectue une authentification biométrique de l'interlocuteur via des modèles SpeechBrain ECAPA-TDNN, puis transcrit les paroles via OpenAI's Whisper.

Les réponses contextuelles sont générées par Groq LLaMA 3.1 8B-Instant, pilotant ensuite un réseau génératif de type GAN entraîné sur mesure pour produire des avatars animés et synchronisés sur le mouvement des lèvres (lip-sync) calculés par OpenCV, NumPy et Pillow.

Projet Personnel // IA & Vision Industrielle

Système de Trafic Intelligent

Python YOLOv8 OpenCV Filtre de Kalman PyTorch

Un pipeline complet d'analyse et de surveillance du flux routier urbain en temps réel basé sur des architectures de vision par ordinateur modernes. Le système met en œuvre YOLOv8 pour localiser et catégoriser les véhicules (voitures, camions, bus, motos) couplé à un Filtre de Kalman pour le suivi multiobjet résistant aux masquages.

Calcule la vitesse estimée de chaque unité, analyse l'occupation des voies, produit des scores d'embouteillages et génère des cartes thermiques de densité. Utilisez les onglets ci-dessus pour passer du flux vidéo brut à la visualisation des métadonnées analytiques de l'IA.

Projet Personnel // Vision par Ordinateur

Forest Fire Detection

Python YOLOv8 OpenCV Analyse HSV MOG2

Un système intelligent d'alerte et de détection précoce des feux de forêt et de la fumée sur flux vidéo en temps réel, conçu pour assister la surveillance environnementale et la prévention des feux.

Le système combine des techniques classiques de vision par ordinateur (filtrage de couleur HSV et suivi de mouvements MOG2) pour isoler les régions d'intérêt suspectes, et un réseau de neurones profond YOLOv8 pour détecter avec précision les feux et le départ de fumées avec génération d'alertes automatiques instantanées.

Python YOLOv8 OpenCV Segmentation Profondeur

Projet Personnel // Vision par Ordinateur

Face Emotion Detector

Python OpenCV Deep Learning Classification

Une application industrielle de vision par ordinateur qui détecte les visages humains depuis un flux vidéo de webcam en direct et catégorise l'état émotionnel de l'utilisateur en temps réel.

Le système utilise un algorithme robuste de localisation faciale sous OpenCV et un classificateur basé sur un réseau de neurones convolutif (CNN) pré-entraîné pour reconnaître les expressions émotionnelles et afficher instantanément un rectangle de suivi annoté avec le pourcentage d'assurance du modèle.

Projet de Fin d'Études // ISS4U

PFE — Synchronisation Odoo & Spring Boot / Angular

Odoo ERP Spring Boot Angular PostgreSQL API REST Synchronisation

Ce projet de fin d'études (PFE) a été réalisé au sein d'ISS4U dans le but de concevoir et d'intégrer un système robuste de synchronisation du flux de données entre l'ERP/CRM Odoo et une application Spring Boot servant de middleware.

La solution permet d'assurer la cohérence et l'acheminement sécurisé des dossiers au sein de l'institut médical. L'application intègre également un tableau de bord moderne développé sous Angular pour le suivi des transactions et interactions par les utilisateurs.
